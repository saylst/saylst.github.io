常见的负载均衡方式有：

* 热备
* 轮询
* 加权轮询
* Ip_Hash

## 热备
假如有两台服务器A和B，只有A故障后才访问B。

```properties
upstream mysvr { 
    server 127.0.0.1:7878; 
    server 192.168.10.121:3333 backup;  #热备
}
```
